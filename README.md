**版本**：v1.0  
**文档日期**：2026-04-14  
**文档状态**：正式版  

---

## 一、产品概述

### 1.1 产品定位

微博会员装扮图片工作室是一个面向微博**运营人员与设计师**的内部工具平台，提供装扮图片规范审核、AI辅助图片生成与延展、以及审核历史管理能力，旨在提升装扮内容的生产效率和规范符合率。

### 1.2 核心价值

| 价值点 | 说明 |
|--------|------|
| 提效 | 自动化替代人工对图尺寸/避让区域/色值等规范的逐项核查 |
| 降错 | 基于标准规范实时检测，减少因规范不符导致的返工 |
| 沉淀 | 审核历史可查询、导出，支持追溯与汇报 |
| 赋能 | 集成 AI 生成与延展，辅助设计师快速产出素材 |

### 1.3 运行平台

- Web 端（PC 浏览器）
- 本地部署，局域网内访问

## 二、目标用户

| 角色 | 使用场景 |
|------|---------|

| 设计师 | 上传设计稿自审，根据 AI 标注定位问题，使用 AI 生成/延展功能产出素材 |

## 三、功能清单

### P0 核心功能（MVP）

#### 3.1 图片审核（单张 & 批量）

**功能描述**：对上传的装扮图片按对应类型规范进行自动化检测，输出审核报告及可视化标注图。

**支持装扮类型（共 10 种）**：

| 装扮类型 | 标准尺寸 | 格式要求 |
|---------|---------|---------|
| 头像挂件 | 250×268px | GIF/PNG，≤350KB，72px分辨率 |
| 评论气泡 | 210×90px | PNG |
| 聊天气泡 | iOS:174×174px / Android:176×176px | PNG 透明底 |
| 卡片背景 | 441×120px | PNG/JPG |
| 编号卡背 | 441×120px | PNG/JPG |
| 铭牌 | SVIP1-7:149×54px / SVIP8-9:162×54px / SVIP10:172×54px | PNG |
| 被赞动效 | 600×600px | PNG，≤300KB |
| 评论大拇指点赞 | iOS:540×540px / Android:432×432px | PNG |
| 喜欢特权 | 900×222px | PNG/JPG |
| 皮肤套装 | 动态视频:1080×608px / 静态:1000×1000px | MP4/JPG |

**核心检测项**：

- 画布尺寸（宽度/高度，容差0px）
- 文件大小
- 分辨率
- 避让区域（是否有设计元素进入禁止区域）
- 色值匹配（评论气泡底色需符合5色标准色值，容差≤3）
- 功能区域占位检测（V标区域、编号区域、主元素区域等）
- 动画时长与帧速率（GIF/视频）
- 深色模式适配检测

**审核输出**：
- 综合评分（百分制）
- 逐项检测结果（通过/失败 + 实际值 vs 标准值）
- 可视化标注图（问题区域高亮圈注）
- 修改建议（分步骤说明）

**批量审核**：支持一次上传多张图片，按队列依次处理，展示进度与汇总结果。

---

#### 3.2 生成记录（审核历史）

**功能描述**：本地持久化存储所有审核历史，支持查询、筛选与导出。

**功能点**：
- 按文件名/装扮类型/审核ID 关键词搜索
- 按审核结果（全部/通过/失败）筛选
- 查看历史审核详情及标注图
- 单条/批量删除记录
- 导出为 Excel 审核报告（含审核ID、文件名、类型、分数、各检测项结果）
- 下载标注图

---

### P1 增强功能

#### 3.3 图片生成（AI 辅助）

**功能描述**：基于 Google Gemini AI 接口，通过文字描述生成装扮运营图片。

**功能点**：
- 输入中文描述词，AI 自动生成图片
- 支持尺寸选择：方形(1080×1080)、横屏(1920×1080)、竖屏(1080×1920)、手机(750×1334)、小屏(375×667)
- 支持风格选择（写实/插画/…）、色调选择
- Prompt 一键优化（自动补充画质/构图/光影描述词）
- 批量生成模式（同一描述多张输出）
- 生成图下载

---

#### 3.4 图片延展（AI 辅助）

**功能描述**：上传现有图片，AI 智能延展至目标尺寸，适配多种使用场景。

**功能点**：
- 上传源图（拖拽/点击）
- 选择目标尺寸（支持多选）：方形、横屏、竖屏、超清竖屏(1440×2560)、超清横屏(2560×1440)
- AI 自动补全延展内容，保持视觉连贯
- 延展结果下载

## 四、用户流程

### 4.1 图片审核核心流程

```
进入审核页面
  → 选择装扮类型（10种）
  → 上传图片（拖拽或点击，单张/批量）
  → 点击「开始审核」
  → 查看审核结果（评分 + 逐项检测 + 标注图）
  → 查看修改建议
  → [可选] 下载标注图 / 导出Excel报告
  → 结果自动保存至历史记录
```

### 4.2 历史查询流程

```
进入生成记录页面
  → 搜索/筛选历史条目
  → 点击查看详情（复现审核报告）
  → [可选] 导出Excel / 下载标注图 / 删除记录
```

### 4.3 AI 生成流程

```
进入图片生成页面
  → 输入描述词（可选：一键优化Prompt）
  → 选择尺寸、风格、色调
  → 点击「生成图片」
  → 下载生成结果
```

## 五、交互说明

| 场景 | 交互反馈 |
|------|---------|
| 上传图片 | 拖拽区高亮，预览缩略图展示 |
| 审核进行中 | Loading 状态，批量审核显示 X/N 进度 |
| 审核通过 | 绿色标识，评分展示 |
| 审核失败 | 红色标识，逐项标注失败项 |
| 可视化标注 | 在原图上叠加矩形/圆形/箭头标注，不同颜色代表不同问题严重度 |
| 历史为空 | 空态提示引导用户进行首次审核 |
| AI 生成中 | 进度动画，预计等待提示 |

## 七、验收标准

| 功能 | 验收条件 |
|------|---------|
| 图片审核-尺寸检测 | 上传标准尺寸图片审核通过；上传错误尺寸图片显示具体实际值与标准值 |
| 图片审核-避让区域 | 有元素进入避让区域时，审核该项失败并在标注图上圈出 |
| 图片审核-色值检测 | 评论气泡底色在标准色值容差3内时通过，超出时失败并给出色差数值 |
| 批量审核 | 多张图片依次审核，全部完成后展示汇总：总数/通过数/失败数 |
| 历史记录 | 审核完成后自动写入历史，页面刷新后记录仍存在 |
| 历史搜索 | 按文件名搜索可正确过滤结果 |
| Excel导出 | 导出文件包含审核ID、文件名、类型、分数及每项检测结果 |
| AI生成 | 输入描述词后能返回图片（依赖 Gemini API 可用性） |
| 图片延展 | 选择目标尺寸后能返回延展图片 |
